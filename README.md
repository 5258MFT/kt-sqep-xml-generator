# KT-SQEP XML 生成器技能

此技能使 Claude Code 能够根据提供的内容生成 **KT-SQEP 兼容的思维导图 XML 文件**。

## 描述

KT-SQEP 是一款思维导图软件。本技能可将任意树状结构的输入（缩进列表、Markdown 项目符号或描述性层级）转换为可直接导入 KT-SQEP 的有效 `.xml` 文件。该技能遵循 KT-SQEP 所期望的精确 XML 模式，包括正确的实体 ID、关系以及布局坐标。

## 技能触发条件

当您执行以下操作时，Claude 会自动使用此技能：

- 提供思维导图大纲（缩进文本、Markdown 列表或描述性层级）
- 要求**生成**、**创建**或**转换**为 **KT-SQEP XML**（或直接说“思维导图 XML”）
- 请求生成一个可以在 KT-SQEP 中打开的文件

触发示例：

- “为下面的项目计划生成一个 KT-SQEP 思维导图 XML……”
- “将此大纲转换为 KT-SQEP XML 文件”
- “为我的会议记录创建一个思维导图 XML”

该技能**不会**在以下情况触发：

- 读取、验证、合并、解析或分析已有的 XML 文件
- 从 XML 中提取数据
- 将 XML 转换为其他格式（JSON、CSV）
- 有关 XML 或思维导图的通用问题

## 安装

1. 将kt-sqep-xml-generator文件夹放入 Claude Code 项目的 `.claude/skills/` 目录中。
2. **提供参考 XML 文件** – `references/` 目录必须包含至少 2–3 个从 KT-SQEP 导出的有效 XML 文件。这些文件作为精确 XML 模式、字段值和约定的标准依据。技能在生成任何新 XML 之前会先读取这些文件。（本项目默认提供了三个简单的示例以供参考）
